# Venue Submission Checklist

## Required package items
- Main manuscript (`main.tex` or `paper.md` generated to LaTeX).
- Bibliography file (`.bib`) with resolved keys.
- Figure assets referenced by manuscript.
- Supplementary files required by target venue.

## Compliance checks
- Anonymity and double-blind requirements are satisfied.
- Page and format limits are checked.
- No placeholder markers remain (`TODO_*`, `TBD_*`, `[CITATION]`).

## Final gate
- Run `/paper-gate-check`
- Run `/latex-build-check`
- Run `/review-report`
