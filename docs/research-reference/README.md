# Research Reference

This directory is the research-focused replacement for the original game-engine reference area.

It stores stable checklists and conventions used by the workflow.

Files:
- `venue-submission-checklist.md` - venue packaging and anonymity checks.
- `latex-bibtex-reference.md` - LaTeX/BibTeX baseline practices for this template.
- `reproducibility-checklist.md` - minimum reproducibility evidence requirements.
