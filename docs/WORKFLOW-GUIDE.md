# Research Workflow Guide (MVP)

## Phase 1: Idea Framing

- Run `/start`
- Run `/research-idea`
- Decide topic, hypothesis, success metrics

## Phase 2: Environment and Backend Setup

- Run `/setup-researchclaw`
- Ensure `researchclaw` is callable
- Create config from template

## Phase 3: Pipeline Execution

- Run `/research-run`
- Use `run`, `resume`, and `report` flow
- Track outputs in `artifacts/` and `deliverables/`

## Phase 4: Paper Quality Gate

- Run `/paper-gate-check`
- Validate citations, evidence, stats, reproducibility, writing completeness

## Phase 5: Submission Prep

- Export final LaTeX and BibTeX
- Run final manual review for venue requirements
