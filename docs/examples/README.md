# Examples

This directory contains minimal end-to-end conversation examples for the ResearchRocket workflow.

Files:
- `01-topic-framing.md` - from topic hint to hypothesis and success metrics.
- `02-literature-to-experiment.md` - from related-work map to executable experiment plan.
- `03-pre-submit-gate.md` - final submission gate with PASS/CONCERNS/FAIL outputs.

Usage:
- Use these as conversation templates when running slash commands.
- Replace placeholders with your topic, dataset, and target venue.
