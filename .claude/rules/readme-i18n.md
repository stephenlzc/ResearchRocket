# README I18N Rules

- Any change to `README.md` must update all localized README files in the same commit.
- Every localized README must include a language navigation section linking all supported languages.
- Command names, agent names, and path examples must remain consistent across languages.
- Translation may adapt wording, but must not change technical behavior or policy meaning.
- Localization must use native, professional terminology instead of literal machine translation.
