# Agent Roster (Extended)

- `principal-investigator` - strategy, scope, and final decisions
- `literature-analyst` - related work map and novelty positioning
- `data-engineer` - data intake, cleaning, schema, and lineage
- `experiment-engineer` - experiment implementation and tracked runs
- `quant-analyst` - analysis plan, statistics, uncertainty, robustness
- `figure-lead` - publication figures and visual traceability
- `writing-lead` - draft structure, argument flow, and claim clarity
- `reproducibility-auditor` - independent reruns and artifact audit
- `submission-manager` - venue packaging and LaTeX/BibTeX checks
- `peer-reviewer` - final PASS/CONCERNS/FAIL recommendation
