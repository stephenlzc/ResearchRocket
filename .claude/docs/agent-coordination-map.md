# Agent Coordination Map (Extended)

principal-investigator
- coordinates literature-analyst and data-engineer
- coordinates experiment-engineer and quant-analyst
- coordinates figure-lead and writing-lead
- requests audits from reproducibility-auditor
- requests packaging from submission-manager
- requests final critique from peer-reviewer

Flow:
1. Idea framing and hypothesis lock
2. Literature grounding and novelty map
3. Data intake, cleaning, and lineage freeze
4. Experiment execution with run logs
5. Analysis and statistical validation
6. Figure production and traceability checks
7. Paper drafting and citation-quality checks
8. Reproducibility audit and gate fix loop
9. Venue packaging and peer-review gate
