# Available Skills (Extended)

- `/start` - research onboarding and project stage check
- `/setup-researchclaw` - setup and validate AutoResearchClaw backend
- `/research-idea` - structured idea generation and hypothesis shaping
- `/research-run` - run/resume/report AutoResearchClaw workflows
- `/paper-gate-check` - pre-submission quality and integrity checks
- `/data-intake` - register sources and create reproducible data intake manifests
- `/data-clean` - normalize and validate raw data into curated datasets
- `/analysis-run` - execute metrics, uncertainty, and robustness analysis workflow
- `/figure-build` - generate publication-ready figures from analysis outputs
- `/figure-qc` - verify figure traceability, captions, and consistency
- `/repro-audit` - independently rerun and audit key experimental results
- `/review-report` - produce PASS/CONCERNS/FAIL review report with blockers
- `/citation-audit` - detect citation placeholders and key-reference mismatches
- `/latex-venue-pack` - prepare venue-aligned LaTeX submission package
- `/latex-build-check` - run LaTeX/BibTeX build sanity checks
- `/paper-style-sanity` - warn-only style checks for vague or AI-slop phrasing

Notes:
- External skills mentioned in ecosystem posts are reference-only in this repository.
- This repository does not copy third-party skill text directly.
